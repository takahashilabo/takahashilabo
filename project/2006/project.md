# ゲームボーイアドバンスを用いたソフトウェア開発演習

任天堂ゲームボーイアドバンス上で自作プログラムが動作する環境を利用して、チームでソフトウェア開発しました。

以下はシミュレータ上で撮影したスクリーンショットです。ゲーム機はフォントデータを持たないため画像データを作成するなど開発に独特の苦労がありました。

2006年は「栄養バランス 管理ソフトの作成」でした。
