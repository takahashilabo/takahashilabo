# Spherotoon：ボール型ロボットによる対戦ゲームの開発

市販品のボール型ロボット（Sphero）を利用して対戦ゲームを開発しました。Spheroはスマートフォンでコントロールします。Spheroの動きはMicrosoft社のKinectを使って追跡します。

開発した作品を山口県で開催されたYMMF2015と小倉で開催されたMONOCAFE2015に出展して来場者の方に遊んでいただきました。
